# 评分与风险规则

## 适岗度评分

适岗度采用 5 个维度的加权评分，输出 0-100 分。

| 维度 | 权重 | 含义 |
| --- | ---: | --- |
| 任务完成度 | 30% | 是否按阶段完成岗位任务 |
| 导师评价 | 25% | 导师对专业表现与协作表现的综合评价 |
| 学习主动性 | 20% | 是否主动学习、提问、同步进展 |
| 沟通协作 | 15% | 与导师和团队的协作质量 |
| 岗位技能匹配 | 10% | 当前能力与岗位要求的匹配程度 |

等级划分：

| 分数 | 等级 | 解释 |
| ---: | --- | --- |
| 85-100 | 高潜 | 可重点培养，具备转正潜力 |
| 70-84 | 稳定 | 表现正常，可继续观察 |
| 60-69 | 需关注 | 存在短板，需要导师干预 |
| 0-59 | 高风险 | 可能不适岗，需要 HR 跟进 |

页面展示中会进一步拆解每个维度：

- 原始分：该维度的 0-100 分。
- 权重：该维度在总分中的占比。
- 贡献分：原始分 × 权重。
- 状态：优势、稳定、需关注、短板。
- 解释：该维度对后续培养动作的含义。

## 风险识别

风险识别采用规则评分 + 导师反馈关键词混合判断。

- `fit_score < 60` 或 `task_score < 50`：高风险。
- `fit_score < 70` 或导师反馈命中风险关键词：需关注。
- 其他情况：低风险。

当前风险关键词：

`偏慢`、`不足`、`迷茫`、`沉默`、`不完整`、`不够稳定`、`拖延`、`缺席`、`质量较低`

## 行动建议

- 高风险：HR 介入沟通、导师拆解明确小任务、实习生提交问题清单。
- 需关注：导师明确验收标准、实习生提高同步频率、HR 观察一周后复盘。
- 低风险：保持节奏，增加真实业务任务与成果沉淀。
