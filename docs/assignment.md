# 作业题目

## 「实习能量站」业务部新人成长导航智能看板

### 一、背景

某部门近期迎来了 20 名新入职的校招实习生，分布在研发、产品、销售等不同岗位。你发现导师（业务部门员工）往往凭经验带教，缺乏标准化的节奏把控；同时，实习生对“我到底该学什么”感到迷茫，频繁私聊 HR 询问进度；此外，招聘同学也常来追问“这批人最近适岗情况如何”。

### 二、任务

请你借助 AI 工具，设计一套“业务部实习生成长导航”智能工具，以解决导师规范带教问题、实习生的成长迷茫感，并提高不同角色之间的协同效率。

### 三、交付物清单

1. 可运行的完整 Demo：形式不限（网页 / 小程序 / 原型 / 智能体 / 多维表均可），须部署至公网并提供可访问链接。
2. 方案说明（1000 字以内）：包括但不限于问题诊断、方案设计、AI 工具选型理由、关键配置、迭代记录与效果评估等，支持 DOC / PDF 格式。
3. 演示视频（非必需）：时长不超过 3 分钟，文件不超过 1GB，支持 MP4 / MOV 格式。
